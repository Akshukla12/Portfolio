# Resume PDF

Place your resume PDF file here.

**Filename:** `resume.pdf`

**Requirements:**
- Professional, up-to-date resume
- PDF format (for universal compatibility)
- Size: < 2MB
- Include: Contact info, experience, education, skills, projects

**Tips:**
- Keep it to 1-2 pages
- Use a clean, professional template
- Ensure it's ATS-friendly (Applicant Tracking System)
- Match the style/tone of your portfolio
- Update regularly

**Privacy note:**
- Remove sensitive information if deploying publicly
- Consider using a contact form instead of phone number
