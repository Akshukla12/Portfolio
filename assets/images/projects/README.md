# Project Screenshots

Place your project screenshots and images here.

**Naming convention:**
- `project-1.jpg`, `project-2.jpg`, etc.

**Recommended specifications:**
- Dimensions: 1200x800px (3:2 aspect ratio)
- Format: JPG or WebP
- Size: < 300KB per image (optimized)
- Quality: High-quality screenshots showing your work

**What to include:**
- Desktop view of your projects
- Key features highlighted
- Clean, professional presentation
- Actual working projects (not mockups)

**Optimization:**
- Use TinyPNG, Squoosh, or ImageOptim
- Consider responsive images with multiple sizes
- Add descriptive alt text in HTML
