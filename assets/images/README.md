# Profile Image

Place your professional profile photo here.

**Recommended specifications:**
- Filename: `profile.jpg` or `profile.png`
- Dimensions: 400x400px (square)
- Format: JPG or PNG
- Size: < 200KB (optimized)
- Style: Professional headshot with clean background

**Optimization tips:**
- Use tools like TinyPNG or Squoosh to compress
- Consider WebP format for better compression
- Ensure good lighting and professional appearance
