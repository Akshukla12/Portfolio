## üí° Pro Tips

### Content Writing
- **About Me**: 2-3 paragraphs, focus on skills and passion
- **Projects**: Use the Problem-Solution-Outcome format
- **Skills**: List 6-8 core skills, 3-4 learning skills
- **Be specific**: "Built responsive dashboard" not "Made website"

### Image Optimization
- Use TinyPNG.com or Squoosh.app
- Target: < 200KB for profile, < 300KB for projects
- Use WebP format if possible
- Always include width/height attributes

### Performance
- Keep images under 300KB
- Minimize custom fonts (we use 1: Inter)
- Don't add heavy JavaScript libraries
- Test on slow 3G connection

### SEO
- Use descriptive page title: "Your Name - Web Developer"
- Write compelling meta description (155 chars)
- Use descriptive alt text for all images
- Add structured data for better search results

## üÜò Troubleshooting

### Site not loading?
- Check that all files are in correct folders
- Open browser console (F12) for errors
- Make sure JavaScript files are loaded

### Theme toggle not working?
- Check browser console for errors
- Verify `js/theme.js` is loaded
- Clear browser cache

### Mobile menu not opening?
- Check that `js/navigation.js` is loaded
- Verify no JavaScript errors in console
- Test in different browser

### Blog posts not showing?
- Check `assets/data/blog-posts.json` exists
- Verify JSON is valid (use JSONLint.com)
- Check browser console for fetch errors

### Form not submitting?
- Form backend not configured yet (normal)
- See "Setup Form Backend" section above
- Check console for errors
